## Pull the Source Code

Run the following commands outside the Go Modules.

```bash
$ go get github.com/xvrzhao/go-examples
$ cd $GOPATH/src/gtihub.com/xvrzhao/go-examples
```
Or
```bash
$ git clone git@github.com:xvrzhao/go-examples.git
```
